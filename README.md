# quicklook

Welcome to the quicklook wiki!

####BASICS####
*  - 0 or more
+  - 1 or more
?  - 0 or 1 
^  - start
$  - end
[] - one of ...
{n} - match n occurrences 

#####SPECIAL CHARACTERS#####
\d - d for decimal
\D - not decimal
\w - w for word(alphanumeric)
\W - not word
\s - s for space
\S - not space
\b - b for boundary
\B - not boundary

#####COMBINATIONS####
[^] - none of
{m,n} - match m to n occurrences
